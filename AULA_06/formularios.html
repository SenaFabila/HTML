Trabalhando com formulários

Elementos para formulários são volstados à interação com o usuário. Entrada de dados, seleção de opções, processamento para manipulação de dados entre outros.

Formulários no HTLM são somente uma Interface.

Para que o formulário funcione é necessário combiná-lo com usamos Linguagem de Programação.
